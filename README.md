# Black_Friday

A expressão "Black Friday" tem origens no final do século XIX e inicialmente era associada a crises financeiras e congestionamentos nas ruas
causados pelo aumento do tráfego devido às compras de Natal. No contexto do comércio, o termo "black" passou a ser associado ao lucro, indicando 
que as lojas passariam de números vermelhos (prejuízo) para números pretos (lucro) devido ao aumento das vendas durante esse período.
